<h1>📌 Overview</h1>

This project demonstrates network traffic analysis using Wireshark.
The goal was to capture and inspect packets to understand how different network protocols operate.

<h2>🎯 Objectives</h2>
Capture live network traffic<br>
Analyze different protocols<br>
Understand packet structure and communication flow<br>
Identify potential security insights

<h3>🔐 Learning Outcome</h3>

This project helped in understanding how data travels across networks and how tools like Wireshark can be used for troubleshooting and security analysis.
